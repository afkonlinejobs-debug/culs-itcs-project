# Cybersecurity Defenders

A simple educational website showcasing basic cybersecurity principles like Firewalls, Encryption, and Ethical Hacking.

## Deployment Instructions (GitHub Pages)
1. Upload all files to your GitHub repository.
2. Go to **Settings → Pages**.
3. Select **Branch: main** and **/ (root)**.
4. Click **Save** — your site will be live shortly.
